# 转录文本阅读器 UI 优化计划

## 1. 项目概述

### 1.1 目标
将原始的 txt 文本阅读器升级为具有顶级阅读体验的 HTML 转录文本阅读器，去除冗余信息，优化信息呈现，提升用户阅读心流和使用体验。

### 1.2 核心痛点
- **信息过载**：原始格式包含语言、处理引擎等无关元数据
- **视觉混乱**：无结构化的纯文本，难以快速定位内容
- **阅读体验差**：缺乏时间戳关联、说话人区分、搜索功能
- **交互缺失**：无法搜索、复制、导出、跳转时间戳

---

## 2. 设计系统

### 2.1 视觉风格定位
- **风格**：现代极简 + 深色沉浸式阅读（Dark Mode First）
- **调性**：专业、专注、内容优先
- **参考**：Apple News、Medium、Notion 的深色模式

### 2.2 色彩系统
```css
:root {
  /* 背景层级 */
  --bg-body: #0A0A0A;        /* 页面底层 */
  --bg-panel: #141414;       /* 主容器 */
  --bg-hover: #1F1F1F;       /* 悬停状态 */
  --bg-active: #2A2A2A;      /* 激活状态 */
  
  /* 文字层级 */
  --text-primary: #E8E8E8;   /* 主标题、说话人 */
  --text-secondary: #A0A0A0; /* 正文内容 */
  --text-tertiary: #6B6B6B;  /* 时间戳、元信息 */
  
  /* 强调色 */
  --accent-primary: #3B82F6;   /* 主强调色 */
  --accent-secondary: #60A5FA; /* 次强调色 */
  
  /* 边框与分隔 */
  --border-subtle: #2A2A2A;
  --border-hover: #3A3A3A;
  
  /* 语义色 */
  --success: #10B981;
  --warning: #F59E0B;
}
```

### 2.3 字体系统
- **主字体**：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
- **等宽字体**："SFMono-Regular", Consolas, "Liberation Mono", Menlo, monospace（时间戳）
- **层级**：
  - 文件名：16px, font-weight: 600
  - 元信息：13px, font-weight: 400
  - 说话人：16px, font-weight: 600
  - 正文：16px, font-weight: 400, line-height: 1.8
  - 时间戳：13px, font-weight: 500

### 2.4 间距系统
- 8px 基础单位
- 段落间距：24px
- 内容区内边距：40px 60px（桌面）/ 20px（移动端）
- 工具栏高度：64px

## 3. 组件架构

### 3.1 整体布局
```
┌─────────────────────────────────────┐
│  工具栏 (Toolbar)                    │
│  ├── 文件信息区                      │
│  │   ├── 文件图标                    │
│  │   ├── 文件名                      │
│  │   └── 精简元信息 (时长/字数)       │
│  └── 操作按钮区                      │
│      ├── 搜索按钮                    │
│      ├── 复制按钮                    │
│      └── 导出按钮                    │
├─────────────────────────────────────┤
│  内容区 (Content Area)               │
│  ├── 转录段落 1                      │
│  │   ├── 时间戳胶囊                  │
│  │   └── 文本内容                    │
│  │       └── [说话人]: 内容          │
│  ├── 转录段落 2                      │
│  │   ├── 时间戳胶囊                  │
│  │   └── 文本内容                    │
│  └── ...                             │
└─────────────────────────────────────┘
```

### 3.2 组件详细设计

#### 3.2.1 工具栏 (Toolbar)
- **定位**：sticky top: 0，z-index: 10
- **背景**：半透明毛玻璃效果（backdrop-filter: blur(12px)）
- **高度**：64px
- **内容**：
  - 左侧：文件图标 + 文件名 + 精简元信息（仅保留时长、字数）
  - 右侧：搜索、复制、导出按钮组
- **滚动行为**：滚动时增加底部边框阴影，增强层级感

#### 3.2.2 搜索功能 (Search)
- **触发**：点击搜索按钮或按 Ctrl/Cmd + F
- **搜索框**：从工具栏展开的输入框，带关闭按钮
- **高亮**：匹配文本高亮显示（黄色背景 + 深色文字）
- **导航**：上下箭头切换匹配项，显示 "3/12" 计数器
- **清空**：一键清空搜索内容

#### 3.2.3 转录段落 (Transcript Segment)
- **布局**：Flex 横向布局，时间戳左挂
- **时间戳**：
  - 胶囊样式：圆角 6px，等宽字体
  - 颜色：主强调色，hover 时背景变浅
  - 点击：可跳转（预留功能接口）
- **文本内容**：
  - 说话人：加粗，主文字色
  - 正文：次文字色，行高 1.8
- **悬停效果**：整段背景色变亮，提升可读性

#### 3.2.4 说话人标识 (Speaker)
- **样式**：font-weight: 600，颜色区分不同说话人
- **扩展性**：支持为不同说话人分配不同颜色标识

## 4. 交互设计

### 4.1 核心交互

| 功能 | 触发方式 | 行为 |
|------|----------|------|
| 搜索 | 点击搜索按钮 / Ctrl+F | 展开搜索框，高亮匹配文本 |
| 复制全文 | 点击复制按钮 | 复制纯文本到剪贴板，显示成功提示 |
| 导出 | 点击导出按钮 | 支持导出 .txt / .srt / .json 格式 |
| 时间戳跳转 | 点击时间戳 | 预留接口，可关联音频播放器 |
| 段落高亮 | 鼠标悬停 | 段落背景色变亮，提升聚焦 |

### 4.2 搜索交互流程
```
1. 用户触发搜索 (点击/Ctrl+F)
2. 工具栏展开搜索输入框，自动聚焦
3. 用户输入关键词，实时高亮所有匹配项
4. 显示匹配计数器 (如 "3/12")
5. 提供上下箭头导航匹配项
6. 点击关闭或按 ESC 退出搜索
```

### 4.3 复制功能流程
```
1. 用户点击复制按钮
2. 提取所有段落纯文本（去除 HTML 标签）
3. 复制到剪贴板
4. 按钮显示 "已复制" 状态（绿色勾选图标）
5. 2 秒后恢复原状
```

### 4.4 导出功能流程
```
1. 用户点击导出按钮
2. 展开下拉菜单：TXT / SRT / JSON
3. 用户选择格式
4. 生成对应格式文件
5. 触发浏览器下载
```

## 5. 响应式设计

### 5.1 断点设计
- **桌面端**：> 768px，完整布局
- **平板端**：768px，调整内边距
- **移动端**：< 640px，简化布局

### 5.2 移动端适配
- 工具栏按钮简化（仅显示图标）
- 转录段落改为垂直布局（时间戳在上）
- 减小内边距（20px）
- 搜索框全宽展开

## 6. 动画与微交互

### 6.1 过渡动画
```css
/* 段落悬停 */
transition: background-color 0.2s ease;

/* 按钮悬停 */
transition: all 0.2s ease;

/* 搜索框展开 */
transition: width 0.3s ease, opacity 0.3s ease;

/* 工具栏滚动阴影 */
transition: box-shadow 0.3s ease;
```

### 6.2 微交互细节
- 按钮悬停：背景色变化 + 文字色变化
- 时间戳悬停：透明度变化 + 背景色变化
- 段落悬停：背景色从透明过渡到浅色
- 搜索匹配：平滑滚动到目标位置

## 7. 数据过滤与清理

### 7.1 去除的冗余信息
- 处理引擎信息（如 "Whisper", "Google Speech-to-Text"）
- 语言代码（如 "en-US", "zh-CN"）
- 置信度分数
- 原始音频元数据（采样率、编码格式等）

### 7.2 保留的核心信息
- 文件名
- 音频时长
- 转录总字数
- 时间戳
- 说话人标识
- 转录文本内容

## 8. 技术实现要点

### 8.1 HTML 结构
- 语义化标签：header, main, section, article
- ARIA 标签：aria-label, role="button", aria-live="polite"
- 无障碍：键盘导航、焦点管理、屏幕阅读器支持

### 8.2 CSS 技术
- CSS 变量：统一主题管理
- Flexbox：段落横向布局
- backdrop-filter：毛玻璃效果
- @media：响应式适配
- ::-webkit-scrollbar：自定义滚动条

### 8.3 JavaScript 功能
- 搜索高亮：正则匹配 + DOM 操作
- 复制功能：Clipboard API
- 导出功能：Blob + URL.createObjectURL
- 滚动监听：工具栏阴影变化
